Welcome to Pokémon Go Norwich

The unoficial companion site for Pokémon Go in Norwich

Gyms and Pokéstops

A map of all the gyms and pokéstops in and around Norwich.

With a search bar so you can find gyms and pokéstops mentioned on Discord or FaceBook.

Nest Map

A map of all the reported nests around Norwich.

You can report nests on Discord, or by selecting a nest and clicking 'Add a Report'.

GitHub

Check out the code that makes this site tick.

This is also the right place to report any bugs, issues or suggestions for the site.
